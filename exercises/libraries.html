  2. define a render() method, accepting an options object with data and el

Define a Chart component

  1. import the Chart service
  2. make its template a single svg node
  3. use ElementRef to get a reference to the root node
  4. call render() and ensure we see some data
    1. data: as with some dummy data (e.g [1,2,3])
    2. el: as our elementRef, .nativeElement.querySelector("svg")
  5. define an @Input() to accept our data once we've loaded it
  6. replace your stubbed call with a call inside an ngOnChanges callback, passing in our data input
  7. now, once we change .data, we should see a rendered chart!

    1. assign a .data property, with stubbed data again

  4. let's pass in real data
    1. in the App handler for our Reddit request, reassign the .data property
