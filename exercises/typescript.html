TypeScript

How-to

All TypeScript work should be written in angular-2-exercise/typescript-exercise.

Run npm run exercise to start an interactive session. Each change you make to a TypeScript file will recompile your code and run the exercise module.

Write lots of console.log()s messages to yourself to test your understanding.

Steps

  1. typescript-vars
    1. in: exercises/typescript.ts
    2. define a const as a number, and assign it a string literal. What happens?
    3. define a const as an array of numbers, and assign it an empty array
      1. then assign it an array of strings
    4. define an object type with some properties. Create some vars with that type. Try and trick the compiler
    5. define some functions with:
      1. typed params
      2. typed return types
  2. import-export
    1. in: exercises/typescript.ts and exercise/module.ts
    2. import Greeter from module - what compiler errors do you get?
    3. define a Greeter class in module.ts
    4. try out a few more import/exports
  3. classes
    1. in: exercises/typescript.ts and exercise/module.ts
    2. in module.ts, define a Header class, with 2 public properties
    3. import into typescript.ts, and create an instance
    4. how does the type checking feel? is it good at inferring things?
    5. define some private properties, and try to access them elsewhere
